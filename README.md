## 🎵 What is SAH?

SAH is the first Web3 platform merging **professional AI music production tools** with a **real utility token economy**. Built for artists, producers, and investors — SAH lets you create, earn, and grow in a self-sustaining ecosystem.

### The Two Pillars

| Pillar | Description |
|--------|-------------|
| **🎹 The Studio** | AI-powered music production tools — stem separation, audio restoration, MIDI transcription, vocal synthesis, and global distribution. Every tool powered by **$SAH**. |
| **⛓️ The Economy** | $SAH is a utility and governance token on **BNB Chain**. Earn by completing tasks, spend on services, and participate in burn decisions that control supply. |

## 🚀 Quick Start

### 1. Open the Mini App
Search for **@Sah_Sonic_bot** on Telegram and tap **Start**. You'll receive **400 SAH** as an early adopter reward.

### 2. Use AI Music Tools
Upload any audio file and use our AI tools:
- **Stem Separation** — Isolate vocals, drums, bass, instruments
- **Audio Restoration** — Remove noise, echo, and artifacts
- **MIDI Transcription** — Convert melody to MIDI files
- **Smart Mastering** — One-click professional mastering

### 3. Earn & Grow
Complete daily tasks, earn SAH tokens, and climb the leaderboard.
